# Basemap Generator


## Data sources:
- Borough Boundaries: [NYC Open Data: Bourough Boundaries](https://data.cityofnewyork.us/City-Government/Borough-Boundaries/yqww-f9f3)
- Subway Lines: [NYS Open Data: MTA Subway Service Lines Map](https://data.ny.gov/Transportation/MTA-Subway-Service-Lines-Map/vtcn-vhd8)
- Building Footprints: [NYC Open Data: BUILDING](https://data.cityofnewyork.us/City-Government/BUILDING/5zhs-2jue/about_data)
- Pavement Edge: [NYC Open Data: NYC Planimetric Database: Pavement Edge](https://data.cityofnewyork.us/Environment/NYC-Planimetric-Database-Pavement-Edge/x9uq-u3qs)
- Sidewalks: [NYC Open Data: NYC Planimetric Database: Sidewalk](https://data.cityofnewyork.us/City-Government/NYC-Planimetric-Database-Sidewalk/vfx9-tbb6)
- Pluto (land use): [NYC Planning Datasets: MapPLUTO](https://www.nyc.gov/content/planning/pages/resources/datasets/mappluto-pluto-change)
    - Download *MapPLUTO - Shoreline Clipped (fgdb)*
- Parks: [NYC Open Data: Parks Properties Map](https://data.cityofnewyork.us/Recreation/Parks-Properties-Map/krz2-j7bn)
- Streets: [NYC Planning Datasets: Digital City Map (DCM)](https://www.nyc.gov/content/planning/pages/resources/datasets/digital-city-map)
    - Download *Digital City Map - ESRI File Geodatabase*
- Zoning: [NYC Open Data: Zoning GIS Data: Geodatabase](https://data.cityofnewyork.us/City-Government/Zoning-GIS-Data-Geodatabase/mm69-vrje/about_data)
